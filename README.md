# timew-fish-plugin
fish plugin with timewarrior completions and functions

## Shell completions

The completion script (`completions/timew.fish`) is taken from [pfmephisto/timew-fishcompletion](https://github.com/pfmephisto/timew-fishcompletion) which is released under [MIT license](https://github.com/pfmephisto/timew-fishcompletion/blob/main/LICENSE).

Issues and pull-requests regarding this, should go there.

## Configuration

- abbreviation `tw` for `timew`
